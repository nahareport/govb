# govb
amazonの商品ページからvaluebooksに行けるブックマークレットです。完璧でありません。
積読チャンネルで紹介された本については、表示されたValueBooksの検索結果ページの下の方をみると在庫があるかもです。

## ブックマークレットの使い方

1. govb.jsのコードをコピーします。
2. ブラウザのブックマークバーに新しいブックマークを作成。
3. 名前は「ValueBooksで検索（ASIN）」など自由に設定。
4. ブックマークの編集画面などから、そのブックマークのURLを1.でコピーした内容に置き換え。 
5. αmaz◯nの商品ページを開いている状態で、そのブックマークをクリックすると、ValueBooksの検索結果ページが新しいタブで開きます。
